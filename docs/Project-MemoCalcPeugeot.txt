Média Móvel (Moving Average):
O que é: A média móvel suaviza as flutuações de curto prazo, destacando tendências de longo prazo.
Como usar: Calcule a média móvel para um determinado período (por exemplo, 3 anos) para suavizar os efeitos de variações extremas.
Benefícios: Reduz o impacto de picos isolados e fornece uma visão mais clara da tendência subjacente.

Desvio Padrão e Análise de Outliers:
O que é: O desvio padrão mede a dispersão dos dados em relação à média.
Como usar: Calcule o desvio padrão para identificar anos atípicos (outliers) que se desviam significativamente da média.
Benefícios: Ajuda a identificar e, possivelmente, excluir picos anômalos para uma análise mais consistente.

Análise de Tendência Linear:
O que é: Ajustar uma linha de tendência linear aos dados para analisar a trajetória geral.
Como usar: Use métodos estatísticos como regressão linear para traçar uma linha de tendência ao longo dos anos.
Benefícios: Fornece uma visão clara da direção geral dos dados, minimizando o impacto de flutuações extremas.

Variação Percentual Anual Composta:
O que é: Calcula a taxa de crescimento composta anual ao longo do período.
Como usar: Considera o valor inicial e final de um período, mas ajusta para o número de anos, suavizando flutuações.
Benefícios: Oferece uma taxa de crescimento ajustada que pode ser mais representativa do desempenho geral.

Análise de Cenários:
O que é: Simule diferentes cenários removendo ou ajustando anos de pico.
Como usar: Recalcule as variações excluindo anos extremos (como 2022 no seu caso) para ver como isso afeta a posição da empresa.
Benefícios: Ajuda a entender como eventos isolados impactam a análise e se a empresa realmente deveria estar em "Top Negativas".
